Foldingathome

https://github.com/FoldingAtHome/coronavirus

Summary

This repository will contain all input files and generated datasets for the Folding@home efforts to better understand how the SARS-CoV-2 virus that causes COVID-19 can be targeted with small molecule and antibody therapeutics.

Type of product: Research

Platform: Desktop client

Initial audience: Medical researchers

Author: Academic coalition

Link to live product: http://foldingathome.org

Main point of contact: g.bowman@wustl.edu

Link to community support: https://foldingforum.org/ https://twitter.com/foldingathome
